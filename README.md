SmartStock - Gestor Inteligente de Inventário

O Problema
    Pequenas e médias empresas do setor varejista frequentemente sofrem perdas financeiras devido à má gestão de estoque. A falta de visibilidade sobre datas de validade e níveis críticos de produtos resulta em desperdício de mercadorias vencidas e perda de vendas por falta de reposição.

A Solução
    O SmartStock é uma plataforma web responsiva para controle rigoroso de entrada e saída de mercadorias. O sistema monitora lotes por data de vencimento e gera alertas preventivos visuais (Dashboard) para otimizar o giro de estoque, permitindo que o gestor tome decisões antes que o produto vença.

Tecnologias Utilizadas
    Frontend: React.js
    Backend: Node.js com Fastify
    Banco de Dados: PostgreSQL
    Autenticação: JWT
    Gestão/Metodologia: Kanban
