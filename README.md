SmartStock - Gestor Inteligente de Inventário

O Problema Pequenas e médias empresas do varejo têm perdas financeiras por causa da má gestão de estoque. A falta de visibilidade sobre datas de validade e baixos níveis de estoque de produtos resulta em desperdício de mercadorias vencidas e perda de vendas por falta de reposição. O SmartStock surge para automatizar esse controle, substituindo planilhas manuais e cadernos por uma interface centralizada.

A Solução Desenvolver um sistema web completo que permita o controle rigoroso de entrada e saída de produtos, monitoramento de lotes por data de vencimento e geração de alertas preventivos para otimizar o giro de estoque.

Tecnologias Utilizadas 
Frontend: React.js 
Backend: Node.js com Fastify 
Banco de Dados: PostgreSQL 
Autenticação: JWT 
Gestão/Metodologia: Kanban
